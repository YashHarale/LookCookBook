Features

Everything you need to know about your next cook.

Search Over Thousands Of Recipes

Welcome to our recipe website, your go-to source for browsing thousands of recipes from all around the world! Whether you're an experienced home cook or just getting started, we have a wide range of recipes to suit your needs, preferences, and tastes.

Add Your Own Recipes

We provide a feature to add your own recipes into our database so that you can share your recipe with other people using the application. You can add different quantity servings depending upon your own need and creativity.

Bookmark Your Favouite Recipe

We also provide a feature to bookmark your Favorite recipes and thereby store them into the storage for further use.

Technology Used

Everything as simple as possible, but no simpler.

HTML, CSS, SASS, ESJS, Bootstrap, etc.

This web application is created using various frontend technologies and tools such as HTML, CSS, SASS, JS, etc. The core application consist of various UI Interactivity features such as Pagination, Lazyloading, Asynchronous loading of recipes in the main container etc.

Node.js, Express.js, MongoDb, Mongoose, etc

The backend of the entire application is build with the help of Js runtime Node.js, Express.js server library, MongoDb Nosql database, Mongoose for connecting the express app with database and nodemon to restart the server. The Signup data of the new user is collected in our database.

Parcel, NPM, Netlify, Git and Github.

Here, we have also used Version control system Git and use service of it i.e Github to push the code. We have deployed the core application on Netlify with continous integration with Github. Also the package bundler Parcel is used to bundle the core app and lastly we have used NPM package manager to manage different node modules.
